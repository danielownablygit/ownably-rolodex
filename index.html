One-time setup required:
This app uses Google OAuth. To run it locally, you need to register it in Google Cloud Console and replace the CLIENT_ID constant in this file with your own OAuth 2.0 Client ID (Web Application type). Add http://localhost and your file path as authorized origins. Enable the Google Sheets API in your project.
